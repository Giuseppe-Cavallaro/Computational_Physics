scattering:
	contiene file Python per il calcolo di ampiezze di Feynman di vari scattering QED,
	la libreria FourVector.py,
	il file check.py per verificare la correttezza di spinori e vettori

hadron:
	contiene un programma C++ per il calcolo della sezione d'urto 
	di una collisione adronica.
	Comando "make" per compilare il file
	Comando "make run" per compilare ed eseguire il main
	"analysis.py" contiene l'analisi dati

metropolis:
	nella sottocartella "devel" ci sono gli algoritmi di
	Metropolis e HMC, con all'interno anche i file di analisi
	dati "analysis.py" e, in HMC, i vari file per il controllo 
	dei check
